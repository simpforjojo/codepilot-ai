# CodePilot AI

CodePilot AI is a backend-first AI Software Engineering Assistant that can analyze Git repositories, understand codebases, build searchable indexes, and answer questions about source code using Retrieval-Augmented Generation (RAG).

## Features

- Repository cloning
- Repository indexing
- File metadata extraction
- Intelligent chunking
- Repository chat (Coming Soon)
- Documentation generation (Planned)
- Multi-agent architecture (Planned)

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- Redis (Planned)
- Docker
- LangGraph
- OpenAI API
